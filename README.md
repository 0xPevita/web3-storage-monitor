# web3-storage-monitor

> Monitor on-chain storage events for Shelby Protocol on Aptos.

![Python](https://img.shields.io/badge/Python-3776AB?style=flat-square&logo=python&logoColor=white)
![Aptos](https://img.shields.io/badge/Aptos-blue?style=flat-square)

## Usage
```bash
pip install -r requirements.txt

python monitor.py 0xYourAddress --interval 15
python monitor.py 0xYourAddress --limit 5
```
